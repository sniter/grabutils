# Grabutils

Utils and helpers for crawling websites

## Install

```shell script
pip install grabutils
```

## Usage

See tests as usage example

## Install 

```shell script
pip install grabutils[all]
```

For `bs` module:

```shell script
pip install grabutils[bs]
```

For `rest` module:

```shell script
pip install grabutils[rest]
```

For `ssh` module:

```shell script
pip install grabutils[ssh]
```
